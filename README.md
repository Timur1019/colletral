# Corporative - Dashboard Application

Современное веб-приложение для управления корпоративными данными с улучшенным дизайном.

## Структура проекта

```
colletral/
├── index.html          # Главная страница (Dashboard)
├── clients.html        # Страница клиентов
├── styles/
│   ├── main.css        # Основные стили
│   └── table.css       # Стили для таблиц
├── scripts/
│   └── main.js         # JavaScript функциональность
└── README.md           # Документация
```

## Особенности дизайна

- **Современный UI**: Градиенты, тени, плавные переходы
- **Адаптивная верстка**: Работает на разных размерах экрана
- **Темная боковая панель**: Стильная навигация
- **Интерактивные элементы**: Hover эффекты, активные состояния
- **Таблицы с фильтрацией**: Удобная работа с данными
- **Пагинация**: Навигация по страницам
- **Графики**: Визуализация статистики с Chart.js

## Технологии

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- JavaScript (Vanilla)
- Chart.js для графиков
- Font Awesome для иконок

## Запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server

# PHP
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000`

## Страницы

- **Dashboard** (`index.html`) - Главная страница со статистикой
- **Clients** (`clients.html`) - Список клиентов с таблицей

## Цветовая схема

- Основной цвет: `#667eea` (фиолетово-синий)
