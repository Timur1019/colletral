# Инструкция по запуску проекта

## Способ 1: Через локальный сервер (рекомендуется)

### Python 3:
```bash
cd /Users/timur/Desktop/colletral
python3 -m http.server 8000
```

Затем откройте в браузере: **http://localhost:8000**

### Node.js (если установлен):
```bash
cd /Users/timur/Desktop/colletral
npx http-server -p 8000
```

### PHP (если установлен):
```bash
cd /Users/timur/Desktop/colletral
php -S localhost:8000
```

## Способ 2: Прямое открытие файла

Можно открыть `index.html` напрямую в браузере, но могут быть проблемы с некоторыми функциями.

## Проверка работы

1. Откройте браузер
2. Перейдите на `http://localhost:8000`
3. Откройте консоль разработчика (F12 или Cmd+Option+I)
4. Проверьте, нет ли ошибок в консоли

## Возможные проблемы

- **Страница не загружается**: Проверьте, запущен ли сервер
- **Стили не применяются**: Проверьте пути к CSS файлам в консоли браузера
- **JavaScript ошибки**: Откройте консоль разработчика (F12) и посмотрите ошибки

## Остановка сервера

Нажмите `Ctrl+C` в терминале, где запущен сервер.
